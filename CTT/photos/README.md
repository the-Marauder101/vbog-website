# Reinstallation Catalog photos

Upload `.jpg` files here. The Reinstallation Catalog calculator
(`CTT/Reinstallation_Catalog.html`) references them by filename via the
`PRICING` object in the inline script.

Expected filenames (one photo per option; if a file is missing the
modal will show alt text and not break):

## Water heater orientation
- `heater_horizontal.jpg`
- `heater_vertical.jpg`

## Water heater capacity (shown in gallery as alt label only)
- `heater_50l.jpg`
- `heater_80l.jpg`
- `heater_100l.jpg`
- `heater_150l.jpg`
- `heater_200l.jpg`
- `heater_300l.jpg`

the `PRICING` object inside `CTT/Reinstallation_Catalog.html`.
